# Tkinter Calculator
A professional-grade GUI calculator built with Python.

## Key Features
* **Interactive UI:** Built using the Tkinter library with a dark-themed interface.
* **Smart Calculation:** Utilizes Python's logic to evaluate mathematical expressions.
* **Error Handling:** Includes a try-except block to handle invalid expressions without crashing.

## How to Use
1. Clone this repository.
2. Run the `mini_project.py` file using Python.
